# Terminal Helper

Terminal Helper 在同一个仓库中分别维护 macOS 和 Windows 原生版本，各平台独立构建与发布。

## 平台索引

- **macOS** — 已完成并可使用；构建、测试和使用说明见 [`TerminalHelper-mac/README.md`](TerminalHelper-mac/README.md)。
- **Windows 11 x64** — 已实现便携版构建与自动化验证配置，发布就绪状态仍取决于 Windows CI 和人工验收；详见 [`TerminalHelper-windows/README.md`](TerminalHelper-windows/README.md)。

两个平台项目彼此独立；平台专属说明以各自目录中的 README 为准。
